# Склады и товары

REST API для управления складами и товарами с использованием Django Rest Framework.

## Описание проекта

Проект представляет собой REST API для создания, получения, обновления и удаления продуктов и складов. Реализована CRUD-логика (Create-Read-Update-Delete) для всех сущностей.

Основные возможности:
- Управление продуктами (создание, чтение, обновление, удаление)
- Управление складами (создание, чтение, обновление, удаление)
- Поиск продуктов по названиям и описаниям
- Поиск складов по идентификатору продукта
- Поиск складов по названию или описанию продукта

## Модели данных

- **Product**: продукты, которыми торгует компания (название, описание)
- **Stock**: склады, на которых хранятся продукты (адрес)
- **StockProduct**: связь между продуктами и складами (количество, цена)

## Технологии

- Python 3.8+
- Django 3.2.4
- Django REST Framework 3.12.4
- PostgreSQL

## Установка и запуск

1. Клонировать репозиторий:
```
git clone git@github.com:Edo19999/Warehouses-and-goods.git
cd Warehouses-and-goods/stocks_products
```

2. Установить зависимости:
```
pip install -r requirements.txt
```

3. Создать базу данных в PostgreSQL и настроить подключение в `settings.py`

4. Выполнить миграции:
```
python manage.py migrate
```

5. Запустить сервер:
```
python manage.py runserver
```

## API Endpoints

- `GET /api/v1/products/` - получение списка всех продуктов
- `POST /api/v1/products/` - создание нового продукта
- `GET /api/v1/products/{id}/` - получение информации о продукте
- `PATCH /api/v1/products/{id}/` - обновление продукта
- `DELETE /api/v1/products/{id}/` - удаление продукта
- `GET /api/v1/products/?search=query` - поиск продуктов по названию или описанию

- `GET /api/v1/stocks/` - получение списка всех складов
- `POST /api/v1/stocks/` - создание нового склада
- `GET /api/v1/stocks/{id}/` - получение информации о складе
- `PATCH /api/v1/stocks/{id}/` - обновление склада
- `DELETE /api/v1/stocks/{id}/` - удаление склада
- `GET /api/v1/stocks/?products=id` - поиск складов по идентификатору продукта
- `GET /api/v1/stocks/?search=query` - поиск складов по названию или описанию продукта
